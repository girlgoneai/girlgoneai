ADHD Daily Planner & Focus System

A Notion Template Built for ADHD Brains

QUICK START — Read This First (Takes 2 Minutes)

  1. Duplicate this template to your Notion workspace
  2. Open TODAY'S page (it's in the Daily Pages database)
  3. Start with the Brain Dump — just write everything in your head, no filter
  4. Pick your Top 3 for the day (only 3, seriously)
  5. Set your first Focus Session timer

That's it. Don't read the rest yet. Go do something. Come back when you're ready to explore more.

How This Is Different

Most planners were built for neurotypical brains. This one wasn't. No 47-step morning routines. No guilt-trip streak counters. No walls of text. This system works with how your brain actually operates — bursts of energy, hyperfocus spirals, forgetting things exist, and needing a win at every turn.

Rules of this system:

  • You can't fail it. If you skipped a week, just come back. No catching up required.
  • Streaks are optional. Consistency matters more than perfection.
  • Done is better than perfect. A half-filled day page is better than a blank one.

Sensory / Environment Checklist

A dysregulated environment = a dysregulated brain. Check before you start.

Noise

  • Headphones on / noise canceling active
  • Background music / brown noise playing
  • Too loud → find quieter spot or use earplugs
  • Quiet enough as-is

Lighting

  • Natural light (ideal)
  • Bright enough, not glaring
  • Adjusted screen brightness

Physical

  • Fidget tool within reach
  • Comfortable temperature
  • Not hungry
  • Not thirsty (water nearby)
  • Chair / posture okay

Workspace

  • Cleared to working minimum (not perfectly clean, just functional)
  • Phone face down or in another room
  • Notifications off / focus mode on

"Stuck?" Decision Tree

Can't start? Can't continue? Run through this.

Step 1: What type of stuck is this?

  • A) I don't know where to start
  • B) The task feels too big
  • C) I'm bored / avoidant
  • D) I'm overwhelmed / anxious
  • E) My brain just won't engage

If A — Don't know where to start

→ Write one sentence: "The very first physical action for this task is ___."

→ Do only that one action. Nothing else counts yet.

If B — Task feels too big

→ Break it into 3 sub-tasks right now:

  • Sub-task 1:                   
  • Sub-task 2:                   
  • Sub-task 3:                   

→ Start with sub-task 1 only.

If C — Bored / avoidant

→ Option 1: Body double — find someone to work near (in person or via video)

→ Option 2: Race the clock — set a 10-min timer, see how much you can do

→ Option 3: Make it interesting — music, different location, standing up

If D — Overwhelmed / anxious

→ Stop. Do a 5-minute brain dump (write every worry)

→ Then pick ONE thing from the list. Just one.

→ If still overwhelmed, take a 10-min break first — walk, breathe, step outside

If E — Brain won't engage

→ Check the basics: water? food? sleep last night?

→ Try a 5-min "warm-up" task (something very easy and satisfying)

→ Change location — a different chair or room can reset

→ Accept it may be a low-output day. Do what you can. It's enough.

Executive Function Support

Task Initiation Prompts

Before starting a task, answer one of these out loud or in writing:

  • "The first thing I will physically do is..."
  • "I will work on this for just [X] minutes..."
  • "The only thing that matters right now is..."

Transition Cues

Struggling to shift from one task to another?

  • Set a 5-min warning timer before the transition
  • Write one sentence ending the current task: "I stopped here: ___"
  • Say out loud: "I am switching to [next task] now."
  • Do a physical reset: stand up, shake it out, sit back down

Appendix: ADHD Planning Philosophy

Why only 3 priorities?

The research and lived experience of ADHD consistently shows that long to-do lists increase cognitive load, trigger avoidance, and lead to decision paralysis. Three priorities fit in working memory. You can actually decide which to do first without spiraling.

Why celebrate the small things?

ADHD brains are often running on a dopamine deficit. External celebrations — even self-generated ones — literally help regulate the reward system. The celebration section isn't fluff. It's neurological maintenance.

Why is the guide so short?

Because you have ADHD and a long guide is the best way to ensure you never use the system. The goal is to get you doing, not reading about doing.

What if I use this inconsistently?

Good. Inconsistent use of a helpful system beats consistent avoidance of a perfect one. Come back whenever you're ready. The template will be here.
