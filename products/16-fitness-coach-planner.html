    FAQ

    Do I need to know how to use Notion already? Basic Notion familiarity helps, but you don't need to be a power user. The template is fully built out — you just duplicate it to your workspace and start entering your clients. The quick-start guide walks you through every step, including how to customize select fields with your own package types and program names.

    How many clients can I track? There is no practical limit. Notion databases support thousands of rows. Coaches with 5 clients will find it lightweight and easy to maintain; coaches with 60+ clients will benefit from the filtering and sorting views that keep large rosters manageable.

    Can I use this for online clients as well as in-person clients? Yes. The template includes a Coaching Modality field on each client profile (In-Person / Online / Hybrid). Session notes work the same way whether you're logging an in-person session or an async check-in with an online client.

    Does it handle different package types? Yes. The Revenue & Package Tracker supports any package structure — session packs (10, 20, 30 sessions), monthly recurring memberships, and custom arrangements. You define your own package types and pricing in the select fields.

    What if a client is on a program I haven't built yet? You can add programs to the Workout Template Library at any time. Many coaches add their core 3-5 programs at setup and build the library incrementally over time. Clients can be assigned to programs as soon as those programs exist in the library.
